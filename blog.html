Left & Right

Have you ever been been stuck trying to seach or alter a large document whilst using the Shell, without resorting to your favorite programming language? Searching Stackoverflow would most likely have resulted in a one-line solution using either an AWK command as the accepted answer. AWK is a tool that may seem daunting at first, but is so incredibly versatile that you'll wonder why you haven't been using it all along! Although I would typically wrangle data using R or Python, I discovered that this 45 year old command line tool significantly simplified my workflows. Here I'll describe AWK, its syntax and applications. Click here to read the full blog post.
